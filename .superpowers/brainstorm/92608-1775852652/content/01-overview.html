Kilometerregistratie 2.0 — Feature Overzicht

15 upgrades verdeeld over 4 pijlers. Klik op de pijler die je als eerste wilt bespreken.

A

🤖 Automatisering

1. Google Maps afstandsberekening
2. Locatie-autocomplete (historisch)
3. Automatische brandstof-matching (Revolut)
4. Cron voor terugkerende ritten
5. Calendar → rit-suggesties

B

🧠 Slimme Features

6. Locatie-normalisatie
7. Duplicate detection
8. Privé/zakelijk split op werkelijke ritten
9. Verbeterd belastingrapport

C

📊 Analytics & Rapportage

10. Dashboard widget (hoofdpagina)
11. Goal-type analytics (charts, trends)
12. Brandstof-analytics (verbruik/km, trends)
13. Maandelijkse email-samenvatting

D

📱 Mobiele UX

14. Floating quick-entry button + 1-tap logging
15. Km-stand foto-upload (bewijs belastingdienst)

Volgorde-aanbeveling: A (Automatisering) eerst — dit is het fundament waar de rest op bouwt. Maps API + locatie-autocomplete maakt alle andere features slimmer.
