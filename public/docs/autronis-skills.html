Wat is ATLAS?

ATLAS is het operationeel systeem van Autronis. Het verbindt alles: het dashboard, AI agents, skills, hooks, automatisering en projectbeheer in één systeem. Je bedient het vanuit Claude Code met slash commands.

Hoe werkt het?

1
Skills = Slash Commands
Elke skill is een /command die je typt in Claude Code. Bijvoorbeeld /ship om te deployen, /ping Klant voor een klantoverzicht, of /autro om het hele agent team aan te sturen. Skills zitten in ~/.claude/skills/.
2
Agent Team
9 gespecialiseerde AI agents die samenwerken. Autro is de CEO die taken verdeelt. Daan interviewt, Coen codeert backend, Finn bouwt frontend, Wout maakt n8n flows, Bas schrijft prompts, Ari doet research, Leo test, en Gabriel logt alles.
3
Hooks = Automatisch
Hooks draaien automatisch op de achtergrond. Bijvoorbeeld: elke keer dat je een taak afrondt, wordt het dashboard gesynct. Elke keer dat je code commit, draait er een quality check. Je hoeft er niet over na te denken.
4
Dashboard = Bron van Waarheid
dashboard.autronis.nl is waar alles samenkomt: projecten, taken, klanten, financiën, ops room. Skills syncen automatisch met het dashboard.
5
Sessie Flow
Start met /prime of /go. Werk met skills en agents. Bij hoge context: /handoff schrijft alles op. Nieuwe chat: /pickup pakt het op. Klaar: /end sluit alles netjes af.
6
MCP Servers
ATLAS is verbonden met externe tools via MCP: Supabase (database), n8n (workflows), Playwright (browser testing), Discord (communicatie). Agents gebruiken deze automatisch.

Dagelijkse Workflow

Ochtend/prime of /brief — context laden, dagbriefing
Werken/build — taken uit dashboard oppakken en uitvoeren
Bouwen/autro — groot project? Agent team stuurt alles aan
Klanten/ping /mail /invoice — klantwerk afhandelen
Deploy/ship — klaarmaken en live zetten
Einde/end of /handoff — sessie afsluiten, alles gesynct
