Wat is het?

Autronis draait op een zelfgebouwd ecosystem: een centraal dashboard met 48+ features, 95+ database tabellen, multi-agent AI orchestratie, en automatische synchronisatie tussen Mac, PC en telefoon. Geen externe SaaS — alles is eigen code.

Het dashboard (dashboard.autronis.nl) is het zenuwcentrum. Daaromheen draaien projectmappen met Claude Code als development partner, agents die taken uitvoeren, en sync-systemen die alles verbinden. Je praat met de AI, en de AI bouwt mee — met volledige context over het bedrijf, klanten, en projecten.

1 Architectuur — Drie Lagen

1. Context Layer — "Wat weet de AI?"

De AI leest bij elke sessie automatisch bestanden die beschrijven:

  • CLAUDE.md — globale bedrijfsregels, sync-instructies, werkafspraken
  • PROJECT_BRIEF.md — per project: wat het is, waarom, voor wie
  • MASTER_PROMPT.md — context specifiek voor Claude Code sessies
  • TODO.md — AI-gegenereerd stappenplan met fases en taken
  • Auto-memory — patronen die de AI leert over sessies heen (correcties, voorkeuren)
  • Dashboard data — via API: klanten, taken, projectstatus, financiën

2. Build & Deliver Layer — "Hoe bouwt de AI?"

  • Claude Code — AI development partner met directe toegang tot codebase, terminal, en tools
  • Ops Room — 24 AI-agents (builders, reviewers, architects) die taken parallel uitvoeren
  • Dashboard — 48+ features gebouwd in Next.js + TypeScript + Turso
  • Custom Skills — scroll-stop-prompter, SEO strategy, website-builder-3d
  • Hooks — automatische activity reporting en dashboard sync bij elke actie

3. Internal Operations — "Wat draait op de achtergrond?"

  • Project Sync Server — maakt automatisch projectmappen aan bij "Start Project"
  • USB Sync — docs/specs automatisch gesynchroniseerd tussen Mac en PC
  • Screentime Tracker — app-gebruik monitoring, productiviteitsanalyse
  • Dagelijkse Briefing — AI-gegenereerde dagstart met prioriteiten en quick wins
  • Follow-up Automatisering — triggers bij inactieve klanten of vervallen offertes

3 Project Creatie Flow

Van idee tot werkend project — volledig geautomatiseerd:

Idee toevoegen → AI Scoring → Start Project → Map + bestanden → Taken in DB → Notion sync

Bij "Start Project" gebeurt automatisch:

  1. Project record in database aangemaakt
  2. Projectmap ~/Autronis/Projects/[naam]/ aangemaakt
  3. AI genereert PROJECT_BRIEF.md, MASTER_PROMPT.md, RULES.md, TODO.md
  4. Taken worden per fase in de database gezet (claude/handmatig)
  5. Notion pagina gegenereerd
  6. Optioneel: Ops Room agents geactiveerd

Modus: Team

Ops Room agents worden ingeschakeld. Daan doet intake, Jones plant, builders bouwen, Toby reviewt.

Modus: Zelf

Je werkt er zelf aan via Claude Code. Alle context-bestanden zijn direct beschikbaar.

8 Sessie Management

/handoff → Schrijft HANDOFF.md met volledige sessie-staat /pickup → Leest HANDOFF.md, vat samen, gaat verder /commit → Wijzigingen opslaan + dashboard sync

Claude Code waarschuwt automatisch wanneer de context hoog wordt (25%+) en stelt voor om /handoff te doen. De volgende chat pakt naadloos de draad op via /pickup.
