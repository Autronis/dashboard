Werkwijze & Systeem Architectuur

Wat is het?

Autronis draait op een zelfgebouwd ecosystem: een centraal dashboard met 48+ features, 95+ database tabellen, multi-agent AI orchestratie, en automatische synchronisatie tussen Mac, PC en telefoon. Geen externe SaaS — alles is eigen code.

Het dashboard (dashboard.autronis.nl) is het zenuwcentrum. Daaromheen draaien projectmappen met Claude Code als development partner, agents die taken uitvoeren, en sync-systemen die alles verbinden. Je praat met de AI, en de AI bouwt mee — met volledige context over het bedrijf, klanten, en projecten.

1 Architectuur — Drie Lagen

1. Context Layer — "Wat weet de AI?"

De AI leest bij elke sessie automatisch bestanden die beschrijven:

  • CLAUDE.md — globale bedrijfsregels, sync-instructies, werkafspraken
  • PROJECT_BRIEF.md — per project: wat het is, waarom, voor wie
  • MASTER_PROMPT.md — context specifiek voor Claude Code sessies
  • TODO.md — AI-gegenereerd stappenplan met fases en taken
  • Auto-memory — patronen die de AI leert over sessies heen (correcties, voorkeuren)
  • Dashboard data — via API: klanten, taken, projectstatus, financiën

2. Build & Deliver Layer — "Hoe bouwt de AI?"

  • Claude Code — AI development partner met directe toegang tot codebase, terminal, en tools
  • Ops Room — 24 AI-agents (builders, reviewers, architects) die taken parallel uitvoeren
  • Dashboard — 48+ features gebouwd in Next.js + TypeScript + Turso
  • Custom Skills — scroll-stop-prompter, SEO strategy, website-builder-3d
  • Hooks — automatische activity reporting en dashboard sync bij elke actie

3. Internal Operations — "Wat draait op de achtergrond?"

  • Project Sync Server — maakt automatisch projectmappen aan bij "Start Project"
  • USB Sync — docs/specs automatisch gesynchroniseerd tussen Mac en PC
  • Screentime Tracker — app-gebruik monitoring, productiviteitsanalyse
  • Dagelijkse Briefing — AI-gegenereerde dagstart met prioriteiten en quick wins
  • Follow-up Automatisering — triggers bij inactieve klanten of vervallen offertes

3 Project Creatie Flow

Van idee tot werkend project — volledig geautomatiseerd:

Idee toevoegen → AI Scoring → Start Project → Map + bestanden → Taken in DB → Notion sync

Bij "Start Project" gebeurt automatisch:

  1. Project record in database aangemaakt
  2. Projectmap ~/Autronis/Projects/[naam]/ aangemaakt
  3. AI genereert PROJECT_BRIEF.md, MASTER_PROMPT.md, RULES.md, TODO.md
  4. Taken worden per fase in de database gezet (claude/handmatig)
  5. Notion pagina gegenereerd
  6. Optioneel: Ops Room agents geactiveerd

Modus: Team

Ops Room agents worden ingeschakeld. Daan doet intake, Jones plant, builders bouwen, Toby reviewt.

Modus: Zelf

Je werkt er zelf aan via Claude Code. Alle context-bestanden zijn direct beschikbaar.

12 Hooks & Automatisering

PostToolUse → report-activity.sh

Draait async na elke tool-aanroep. Meldt aan de Ops Room welke agent actief is. Mapt project → agent (dashboard → wout, website → pedro, etc.). Real-time zichtbaar in het dashboard.

Stop → sync-dashboard.sh

Bij sessie-einde: herinnering om voltooide taken te syncen naar het dashboard. Check of USB stick gemount is voor docs sync.

Settings.json bevat: hooks configuratie, goedgekeurde commands (git, npm, curl, etc.), en extra directories die Claude Code mag benaderen.

14 Regels & Conventies

Code

  • TypeScript altijd, nooit plain JavaScript
  • Nooit any gebruiken
  • Nooit console.log in productie
  • Code in Engels, UI-teksten in Nederlands
  • Commit messages in Engels
  • Kleine, logische commits
  • Branch per feature

Dashboard Design

  • Dark theme: bg #192225, accent #17B8A5
  • Framer Motion animaties
  • Rounded 2xl cards
  • API responses in Nederlands
  • Factuurnummer: AUT-YYYY-NNN
  • Soft delete via is_actief flag
  • BTW 21%, betalingstermijn 30 dagen

Claude Code Werkwijze

  • GitHub: altijd naar github.com/autronis/
  • Dashboard sync na elke sessie (verplicht)
  • USB sync als stick gemount is
  • Geen losse bestanden in Projects/
  • SpeakToText dictionary bijwerken
  • Beknopt en to-the-point antwoorden

AI Principes

  • Eerst lezen, dan aanpassen
  • Geen extra features die niet gevraagd zijn
  • DRY, KISS — geen speculative abstractions
  • Context waarschuwing bij 25%+
  • Handoff/pickup bij sessie-wissel
  • Auto-memory voor patronen over sessies

17 Wat maakt het anders?

Context-aware AI. De AI weet niet alleen hoe je code schrijft, maar ook wat je bedrijf doet, wie je klanten zijn, wat de prioriteiten zijn, en wat er vorige week is besloten. Het is geen generieke copilot — het is een partner die het hele bedrijf kent.

Eigen platform, geen SaaS-afhankelijkheid. Waar de meeste agencies 10+ losse tools aan elkaar knopen (CRM, facturatie, projectmanagement, content planning), draait Autronis op één zelfgebouwd dashboard met 48+ features. Alles in eigen beheer, alles op elkaar afgestemd.

Multi-agent orchestratie. Niet één AI-assistent, maar 24 gespecialiseerde agents die parallel werken. Intake, planning, bouw, review — elk met een eigen rol en permissie-niveau. De AI is geen tool, het is een team.

Cross-session memory. De AI leert van correcties en onthoudt patronen. Als je zegt "doe dit nooit meer zo", dan doet het dat niet meer — ook niet in de volgende sessie, volgende week. Context gaat nooit verloren.

Volledig geautomatiseerde project pipeline. Van idee → AI scoring → projectmap met bestanden → taken in database → Notion sync. Eén klik op "Start Project" en alles staat klaar.

Anywhere access. Mac op kantoor, PC thuis, telefoon onderweg — via Tailscale VPN, Wake-on-LAN, en SSH kun je overal aan elk project werken met volledige AI-ondersteuning.
